This is my Capstone Project that I created in 2020, after completing my 3-month Data Science Bootcamp. This is Version 2, which has some improvements over the last version, using some of the things I learnt these past couple of years.

Version 1:

  • Run scraping manually everyday.
  • No OOP.
  • Clean and divide data by day.
  • No database, everything was saved in an Excel spreadsheet, then manually merged.
  • Separate operation to train ML models.
  • Basic Flask app.

Version 2:

  • Dockerized application.
  • OOP.
  • Somewhat better Flask app.
  • Better organization and structure.
  • Existing database that will keep a version locally for persistence.

Version 3: (to come)

  • Offload scraping to Airflow for automation.
  • Create workflow to group data by time blocks.
  • Data augmentation and ML model training.
  • Deploy model.
  • Automate ML model re-testing, re-training, and re-deployment in Airflow.
